Welcome to Access Control

AccessControl is a smart lock management system built for both security and speed. It gives you a clear, modern UI for managing Sites, Locks, Cardholders, and Schedulesâ€”and a robust, cloud-ready backend that scales.

Under the hood we run on .NET 8 and Angular + PrimeNG. Commands and queries are cleanly separated with MediatR and CQRS, while MassTransit + RabbitMQ power our event-driven workflows (from live events to async processing). Authentication uses JWT, and data is stored in MartenDB on PostgreSQL for flexible, document-first modeling.

The result: predictable operations, auditable activity, and fast iteration when your access rules evolve.

  • Site: Your physical locations.
  • Lock: Controlled entry points per site.
  • Cardholders: People with permissions.
  • Schedules: Time windows for allowed access.
